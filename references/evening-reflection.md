# Evening Reflection

## Workflow

### Step 1: Load Today's Note
Read `$VAULT/1-Daily/YYYY-MM-DD.md`. If it doesn't exist, create from the daily-note template.

### Step 2: Franklin-Style Self-Examination
Read `references/franklin-reflection.md` for the full question set. Present a concise version:

Ask the user (use AskUserQuestion or direct questions):

1. **What good did I do today?**
   - Key wins and accomplishments

2. **What did I learn today?**
   - New insights, skills, or realizations

3. **What could I have done better?**
   - Mistakes, missed opportunities, improvements

4. **What am I grateful for?**
   - 1-3 specific gratitudes

### Step 3: Task Review
1. Check each MIT from the morning section — mark as done or carry forward
2. Review the task log section — any uncompleted items get noted
3. If tasks were completed today that weren't in the morning plan, note them

### Step 4: Habit Check
If habits are defined in `$VAULT/8-Habits/`, ask the user to confirm today's habit completion. Update the habit log at `$VAULT/8-Habits/YYYY-MM-DD.md`.

### Step 5: Tomorrow Prep
Ask the user: "What's the one thing you want to accomplish tomorrow morning?" Write it to today's evening reflection section under "Tomorrow Prep".

### Step 6: Write Reflection
Append the completed evening reflection to today's daily note, filling in the Evening Reflection section.

## Constraints
- This should feel like a calm wind-down, not an interrogation
- If the user is tired or rushing, offer a "quick mode" with just 2 questions (wins + gratitude)
- Never guilt-trip about incomplete tasks — the purpose is reflection, not judgment
